# Crazy Golf Mobile App

A simple Android app for tracking Crazy Golf scores with persistent storage using SQLite. Supports multiple players, round history, and a clean, easy-to-use interface.

---

## Features
- Add players and record hole-by-hole scores  
- Automatic total score calculation  
- View past game history  
- SQLite database for persistent local storage  
- Input validation and stable UI built with XML  

---

## Tech Stack
- **Java**
- **Android Studio**
- **SQLite**
- **XML (UI layout)**
